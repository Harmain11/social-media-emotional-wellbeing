# Social Media Emotional Wellbeing Analysis

## Overview
This notebook explores the relationship between social media usage and emotional wellbeing using a dataset from Kaggle. It performs data cleaning, visualization, and trains a Random Forest classifier to predict dominant emotions based on social media usage features.

## Features
- Data download and preparation from Kaggle using kagglehub
- Exploratory data analysis with histograms, boxplots, count plots, correlation heatmap, and scatterplots
- Data encoding and feature engineering
- Random Forest classification model training and evaluation
- Visualization of model performance (confusion matrix, classification report, ROC curves)
- Feature importance analysis

## Tech Stack
- Python
- Jupyter Notebook / Google Colab
- pandas
- matplotlib
- seaborn
- scikit-learn
- kagglehub

## How to Use
1. Clone or download this repository.
2. Open the notebook `social-media-emotional-wellbeing-analysis.ipynb` in Jupyter Notebook or Google Colab.
3. Run the cells sequentially to download the dataset, explore data, train and evaluate the model.
